📜 The Rules of the Hive

A Honey Table is a 10-row comb. The first column is its skeleton; the rest of the comb hides one secret word, one letter per column, lined up by the birthday.

The structure column (column 0)

Read top to bottom: the blue top cell is the direction, the red cells hold the 8 key digits (your date as DDMMYYYY), and the green bottom cell is the version. Direction + 8 digits + version = 10 cells, so the comb always has 10 rows.

Placing the word

Each letter gets its own column (column 1, 2, 3 …). The column's header is the matching key digit, and the letter sits at the row equal to that digit. So with key 15081947, letter 1 → row 1, letter 2 → row 5, letter 3 → row 0, letter 4 → row 8, letter 5 → row 1.

Reading it back

Knowing the birthday, walk column by column: in column i jump to the row given by the i-th key digit and lift the letter. A wrong birthday jumps to the wrong rows and pulls out decoys.

Versions

The current version is 0. The red key cells double as room for future versions to change the rules — the comb carries its own version so a reader always knows how to open it.
